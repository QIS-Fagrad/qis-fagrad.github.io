This website is built with [SvelteKit](https://svelte.dev/tutorial/kit/introducing-sveltekit) and developed using [Bun](https://bun.sh).

## Development

1. [Install Bun](https://bun.com/get)

```sh
# Linux/macOS
curl -fsSL https://bun.sh/install | bash
```

```sh
# Windows (PowerShell)
powershell -c "irm bun.sh/install.ps1 | iex"
```

2. Install dependencies

```sh
bun install
```

3. Run locally

```sh
bun dev
```

4. Build and preview the static website locally

```sh
bun run build && bun run preview
```
